# Honesty Boundaries

- ArchitectAnswer is not Evidence.
- EvidenceProbe is a plan, not execution.
- Scope compliance is not correctness.
- Bounded task closure is not full repository understanding.
- Candidates are not active authority.
- Closure reports do not mutate the live graph.
- A complete-looking artifact with unavailable inputs remains incomplete.
- Returning to admission requires an explicit input change, not hope.

When a source is unavailable, stale, unsupported, or outside the bundle, report
that exact limitation and stop at the smallest justified next action.
